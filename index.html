Array Methods

Array metodlari JavaScript tilida massivlar (arrays) bilan ishlashni osonlashtiruvchi vositalardir. Ular yordamida elementlar qo'shish, olib tashlash, qidirish, filtrlash va boshqa ko'plab vazifalar bajariladi.

Array Methods Types

push() Arrayning oxiriga yangi element qo'shadi.

let arr = [1, 2, 3];
arr. push(4); // arr endi [1, 2, 3, 4]

pop() Arrayning oxiridan elementni olib tashlaydi va qaytaradi.

let arr = [1, 2, 3];
let lastElement = arr. pop(); // arr endi [1, 2] va lastElement = 3

shift() Arrayning boshidan elementni olib tashlaydi va qaytaradi.

let arr = [1, 2, 3];
let firstElement = arr. shift(); // arr endi [1, 2] va firstElement = 1

unshift() Arrayning boshiga yangi element qo'shadi.

let arr = [1, 2, 3];
arr. unshift(0); // arr endi [0, 1, 2, 3]

splice() Arraydan elementlarni olib tashlash yoki qo'shish uchun ishlatiladi.

let arr = [1, 2, 3, 4];
arr. splice(2, 1, 'newElement'); // arr endi [1, 2, 'newElement', 4]

slice() Arrayning bir qismini yangi arrayga qaytaradi, original arrayni o'zgartirmaydi.

let arr = [1, 2, 3, 4];
let newArr = arr. slice( 1, 3 ); // newArr = [2, 3]

concat() Ikki yoki undan ortiq arraylarni birlashtiradi va yangi array qaytaradi.

let arr = [1, 2];
let arr2 = [3, 4];
let combinedArr = arr1. concat(arr2); // combinedArr = [1, 2, 3, 4]

forEach() Har bir element uchun funksiya bajaradi.

let arr = [1, 2, 3];
arr. forEach(element => console.log(element)); // 1 2 3

map() Har bir element uchun yangi qiymatga ega bo'lgan yangi array qaytaradi.

let arr = [1, 2, 3];
let newArr = arr.map(element => element * 2); // newArr = [2, 4, 6]

filter() Shart bo'yicha elementlarni filtrlaydi va yangi array qaytaradi.

let arr = [1, 2, 3, 4];
let filteredArr = arr.filter(element => element > 2); // filteredArr = [3, 4]

reduce() Barcha elementlarni yagona qiymatga qisqartiradi.

let arr = [1, 2, 3, 4];
let total = arr.reduce((accumulator, currentValue) => accumulator + currentValue, 0); // total = 10

find() Shartga mos keladigan birinchi elementni qaytaradi.

let arr = [1, 2, 3, 4];
let found = arr.find(element => element > 2); // found = 3

findIndex() Shartga mos keladigan birinchi elementning indeksini qaytaradi.

let arr = [1, 2, 3, 4];
let index = arr.findIndex(element => element > 2); // index = 2

includes() Elementning mavjudligini tekshiradi.

let arr = [1, 2, 3];
let hasTwo = arr.includes(2); // hasTwo = true

sort() Array elementlarini tartiblaydi.

let arr = [3, 1, 4, 2];
arr. sort(); // arr endi [1, 2, 3, 4]

reverse() Array elementlarini teskari tartibda qaytaradi.

let arr = [1, 2, 3];
arr. reverse(); // arr endi [3, 2, 1]
